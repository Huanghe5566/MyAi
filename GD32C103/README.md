# GD32C103 CAN 网关与控制器项目

## 项目简介
本项目基于 **GD32C103** 系列微控制器实现，主要负责高性能 CAN 消息的信号映射、转换、定时诊断上报以及按键状态管理。

## 核心功能
- **CAN 协议转换**：实时将源 CAN 消息映射并转发为目标协议的标准格式。
- **高精度定时器任务**：支持多种周期的并行任务执行。
- **定时诊断报文**：自动周期发送标准 UDS 诊断报文。
- **按键交互**：集成 `MultiButton` 状态机。

## 优化记录
### 2026-04-22 周期漂移修复
- **现象**：0x700 报文实际发送周期为 2082ms。
- **方案**：将 `tick` 修改为计数器模式，并在 `main.c` 中通过循环消费积压的 `tick` 值。
- **结果**：任务周期回归精准的 2000ms。
